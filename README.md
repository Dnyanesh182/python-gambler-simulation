## Gambler Simulation Problem – Use Cases

UC1 – Initialize Stake, Goal, and Bet Amount  
UC2 – Simulate a Single Bet (Win or Loss)  
UC3 – Update Stake Based on Win or Loss  
UC4 – Repeat Betting Until Stake Reaches Goal or Zero  
UC5 – Count Total Number of Bets Made  
UC6 – Count Total Wins and Losses  
UC7 – Calculate Win Percentage  
UC8 – Calculate Loss Percentage  
UC9 – Display Final Gambling Statistics  
UC10 – Allow Multiple Gambling Simulations Using Loop
